SendCharge

Design and development of a highly complex SaaS website for an Australian startup company.

This online software allows customers to compare and book courier companies or freelance delivery drivers online.

Freelance delivery drivers work similar to Uber drivers. This includes GPS tracking, driver profiles, automated quoting, feedback / ratings, socket based chat, real-time tracking updates and more.

This project also includes an advanced backend for users with digital wallets, bulk / automated fulfilment for high volume businesses, 3rd party API integration, advanced email and phone verification, advanced account settings, etc.

This huge piece was delivered entirely by myself and one API developer. Some of my best work to date. A lot of great functionality here and a very clean codebase.

Unfortunately this project is no longer online. The startup eventually lost funding and was sold for investment capital by the stakeholders :(
